Class Action

A wrapper around act to store the operations array. The operartions can be an instance of Lazy so that it is computed every time Action#act is called.

Example

import { Action } from 'apption'
let count = 0;
const action = new Action([
    (a1, a2) => count += a1,
    (a1, a2) => count += a2,
    (a1, a2) => count += a2 + 1
]);
action.act(20, 21);
console.log(count);   // 63
